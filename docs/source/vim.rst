VIM Shortcuts
--------------

.. code-block:: sh

   ZZ  # Save current line, close it
   s   # Delete the current line, go to insert mode
   w   # Move 1 word <-
   b   # Move 1 word ->
   yyp # Duplicate current line

If you open a non-modifiable file with vim
-------------------------------------------
.. code-block:: sh

   :w !sudo tee %
