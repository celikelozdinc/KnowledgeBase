VIM Shortcuts
--------------

Here are some shortcuts:

.. code-block:: sh

   ZZ  #Save current line, close it

   s   #Delete the current line, go to ``"insert"`` mode

   w   #Move 1 word <-

   b   #Move 1 word ->

   yyp #Duplicate current line
